# N&E Softscapes Website

This is a fun and bright website for **N&E Softscapes**, a young landscaping company.  

## Pages

- **Home** – Welcome page with brand colors and logo  
- **Portfolio** – Showcases completed landscaping projects  
- **Contact** – Let's clients get in touch  

## How to view

You can see the live site here: [GitHub Pages Link](https://zohunt.github.io/ne-softscapes/)
